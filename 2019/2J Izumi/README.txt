2J 泉 知成 映像認識防犯システム(仮)について



・現状

　　ラズパイの外付けカメラから得た映像から人の顔を認識することができた(精度いまいち)

　　特定のキーを押したときにのみ写真を撮影し、その画像データをLINEに送信する(←完成)



本当は「顔認証」や「表情からの感情認識」等をやりたかったが、高専プロコン(競技部門)の関係や
その他学校の課題・テスト勉強などで忙しく想定していたよりも開発することができなかった。


・将来的に実装したい機能

　　顔認証でその顔が誰なのかを特定する。

　　表情から感情を読み取る。


(とりあえずオープンキャンパスまでにはLINEの送信機能までを確実に実装するようにする。
)
↑動画を取りつつ並行して画像を撮影し、画像とメッセージをLINEへ送信はできるようにはなったが、
　ラズパイの外付けカメラを上下逆向きに設定してしまったため、
　ラズパイ上で画像を表示する分にはOpenCVの関数で何とかなるが、
　LINEに送信される画像は上下逆向きになってしまうという問題が発覚した。(←時間とメンタルの問題で改善できるかどうかはわからない)

オープンキャンパス本番ではLINEに画像を送信しなくてはならない訳だが、
その場合、ラズパイ本体をネットワークに接続しなければいけないという問題がある。
学校のネットワークには接続できないため、ポケットWi-Fiを持参しなければならなくなると思うが、
何通も何通も画像を送信すると、今月の残り僅かな私の人権(通信可能容量)が底を尽きてしまう可能性があるため
A4サイズのコルクボードにあらかじめ作っておいた説明の用紙を貼り付け、本番は図とともに説明しようかと考えている。
（↑これを採用する可能性大）


オープンキャンパス以降は高専プロコンに使用する遺伝的アルゴリズムを優先して作成するが、
夏休みなど時間が空いているときに開発したいと思う。(できれば高専祭までに...)
